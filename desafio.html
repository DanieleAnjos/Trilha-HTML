História dos computadores

O primeiro computar foi o “ábaco”, instrumento mecânico de origem chinesa criado no século V a.C. Ele é considerado o “primeiro computador”, uma espécie de calculadora que realizava operações algébricas.

Primeira geração

Funcionavam por meio de circuitos e válvulas eletrônicas. Além de possúi o uso restrito, eram imensos e consumiam muita energia.

Um exemplo é o ENIAC primeiro computador eletrônico de grande escala do mundo.

Segunda geração

Os computadores passaram a funcionar por meio de transistores, substituindo as valvulas eletrônicas.

Terceira geração

Computadores começam a usar circuitos integrados, apresentado maior capacidade de processamento.

Nesse periodo começou a produção dos chips e a utilização de computadores pessoais.

Quarta geração até os dias atuais

Com o aprimoramento da tecnologia, os computadores passaram a usar microprocessadores.

Tornando os computadores cada vez menores, com menos gasto de energia e mais eficientes. Apartir daí houve o surgimento dos computadores de mão(smartphones e tablets).

Client e Sever

Um servidor é um recurso dentro de um sistema computaciona, usado para processar informações, armazenar dados etc. Enquanto o client é um computador que solicita os serviços do servidor.

A comunicação cliente-servidor é baseada em troca de mensagens, onde o cliente envia requisições e o servidor responde com as informações requisitadas.

O que é HTML?

HTML (Linguagem de Marcação de Hipertexto), é um código usado para definir a estrutura de uma página web.

Elementos

O HTML é composto por elementos que estruturam e definem o conteúdo de uma página da web.

Tags

As tags e os elementos entre elas são chamadas de elementos do HTML.

  • Tag de abertura: é usada para iniciar um elemento.
  • Conteúdo: Informações(textos e outros elementos) que está entre as tags de abertura e de fechamento.
  • Tag de fechamento: usada para finalizar um elemento.

    • Atributos

    São palavra expecificas usadas dentro da tag de abertura para controlar o comportamento do elemento. Com eles podemos identificar melhor um elemento, e informar qual tipo de arquivo, texto etc...

    Exemplos

    • src= ultilizado para identificar qual tipo de arquivo ou link, a tag deve utilizar.
    • alt="Texto alternativo=" fornece alternativas caso a imagem não seja carregada ou caso o usuário use um leitor de tela.
    • lang="Linguagem" expecifica o idioma do conteúdo da tag.
